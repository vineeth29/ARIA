# AI Project

\# ARIA - AI Desktop Assistant



ARIA is an advanced AI-powered desktop assistant built using Python.



\## Features



\- Voice Assistant

\- AI Chat Support

\- Smart Automation

\- Mood Detection

\- Health Monitoring

\- Clipboard History

\- Screen Watching

\- Smart Scheduler

\- Focus Mode

\- Habit Tracking

\- Window Management

\- AI Personalities

\- Web Search

\- Notes System



\## Technologies Used



\- Python

\- Ollama

\- Groq API

\- Gemini API

\- Cerebras API



\## Installation



Clone the repository:



```bash

git clone https://github.com/vineeth29/ARIA.git

```



Move into the project folder:



```bash

cd ARIA

```



Install dependencies:



```bash

pip install -r requirements.txt

```



\## Configuration



Create your own `config.json` file based on:



```text

config.template.json

```



Add your API keys locally.



\## Run



```bash

python aria.py

```
